# 🔄 Logic Apps & Playbooks

This section demonstrates Security Orchestration, Automation, and Response (SOAR) capabilities implemented using Microsoft Sentinel Automation Rules and Azure Logic Apps Playbooks.

The implemented playbooks automate incident response workflows such as:
- email-based incident notification
- automated incident enrichment
- investigation workflow orchestration

These workflows simulate real-world SOC automation processes used to improve incident response efficiency and reduce manual analyst effort.

---

# 🧠 Technologies Used

- Microsoft Sentinel
- Azure Logic Apps
- Automation Rules
- Office 365 Outlook Connector
- Managed Identity
- Role-Based Access Control (RBAC)

---

# 🚀 Implemented Playbooks

| Playbook | Description |
|---|---|
| Email Notification Playbook | Automatically sends email notifications whenever new Sentinel incidents are created |
| Incident Enrichment Playbook | Automatically enriches Sentinel incidents by adding investigation comments and contextual information |
